Microsoft 下载中心：常见问题解答

+为什么需要更新软件？

Microsoft 一直致力于向其客户提供经过安全测试的软件。尽管没有任何系统是完全安全的，但是我们通过各流程、技术和多个专门团队研究、修复和了解安全问题， 由此帮助我们满足此目标，并向客户提供有关如何帮助保护其 PC 的指导。

作为 Microsoft 软件发布流程的一部分，发布到下载中心的所有软件都经过恶意软件扫描，然后才可供公众下载。另外，发布后，也会定期对下载中心提供的软件进行恶意软件扫描。 Microsoft 认识到威胁环境是不断变化的，并且会随着时间推移而持续演变，因此我们一直在努力改进，以帮助保护客户免遭恶意软件威胁。

+如何使软件保持最新？

Microsoft 提供了一系列联机服务，帮助使您的计算机保持最新状态。Windows Update 会查找您甚至可能未察觉到的更新，并向您提供用于安装更新的最简单方法。 这些更新有助于预防或修复问题，改进计算机的工作方式或增强您的计算体验。请访问 Windows Update 以了解更多信息。

+可以在 Microsoft 下载中心找到什么内容，如何找到我所需的内容？

Microsoft 下载中心最近经过修订，能够以一站式商店形式更好地向您提供服务。此商店中提供了可供购买的产品，以及可免费获得的产品和下载。为方便起见，可供购买的项目将直接链接到 Microsoft 商店。可免费下载的项目将链接到详细信息页，您可以在详细信息页中了解项目详细信息并开始下载。

如果希望 Microsoft 扫描您的计算机并推荐更新，请参阅前面的问题。如果您想要搜索特定的下载，请首先使用下面的一种方法。

  • 要查找产品和下载，请在显示于下载中心主页中央的导航栏中单击相应图标。
  • 要搜索更具体的产品或下载，请在页面顶部附近显示的菜单栏中单击菜单名称，然后单击产品或下载。
  • 要按关键字搜索，请在页面顶部的搜索下载中心搜索框中，键入您要搜索的一个或多个词。

在搜索结果页上，您可能会看到下面的内容：

  • 基于搜索词而建议的产品及下载的编辑者选项列表。
  • 页面右侧的排序依据框。单击向下箭头，选择搜索结果的显示顺序。
  • 页面左侧的筛选器选项列表。单击一个筛选器选项可缩小在搜索结果中显示的项目列表范围。该筛选器选项与原始搜索词可一起缩小搜索结果的范围。

当您看到感兴趣的项目时，可以单击项目链接以了解有关下载和购买该项目的详细信息。

+如何查找全球下载？

Microsoft 在全球以超过 118 种的语言形式提供下载。下载中心现在将所有英语下载合并为一个英语下载中心。我们不再提供美国英语、英国英语、澳大利亚英语或加拿大英语的单独下载。

您可以通过以下两种主要方式查找全球下载：

访问全球下载中心网站
  • 您可以在英语下载中心网站每个页面的页脚处，单击全球下载。
  • 在列表中单击一种语言。您将转到全球下载中心主页，在该页面上，您可以浏览和搜索以所选语言形式提供的下载。

注意： 如果可能，全球下载中心网站上的下载详细信息页会以相应的语言显示。但是，如果不存在非英语的特定语言详细信息，则这些详细信息将以英语显示。如果下载本身未以特定语言形式提供，则它将不会显示在您选择的全球网站上。

从英语语言网站选择全球下载
  • 在特定项目的下载详细信息页上的“快速详细信息”下，您将会看到更改语言框。可从下拉列表中选择所需的语言。

注意： 仅当存在下载的全球版本时，更改语言框才会出现。如果此框未出现，则表示相应下载没有以非英语的语言形式提供。如果您看到了此框，但未列出特定语言，则表示相应下载还没有以该语言形式提供。

+如果找不到所需的内容，我应该怎么做？

如果您找不到某个特定的下载，则表示它可能是由 Microsoft 以外的公司提供。常见的示例包括 Adobe Reader、Macromedia Shockwave 和 Flash Player 以及 Java 软件。 要获取这些产品在各自网站上的当前下载链接，请使用Bing在 Web 上搜索您要查找的下载。如果您确定某特定下载是由 Microsoft 提供的，但您无法在下载中心内或通过我们的自动 更新服务找到该下载，请联系我们 。

+我将在下载页面中找到哪些信息？

当您单击下载中心的一个下载项目时，您将转到该下载的下载详细信息页。这些页面中的大多数都采用一种标准格式，并包含此处描述的大多数部分。

  • 快速详细信息
    快速详细信息部分包含有用的概要信息，包括下载文件的名称和大小及其发布日期。
  • 更改语言
    在某些情况下，您将会看到更改语言框。您可以从该框中选择英语之外的语言。请注意，某些下载并不提供全部的语言形式。另外，并非所有非英语的下载都具有本地化的详细信息页。因此，您可能会在非英语下载的页面上看到英语形式的描述和说明。
  • 下载按钮或链接
    对于每个列出的文件，您将会在快速详细信息部分看到下载按钮或链接。单击此按钮或链接可开始下载。
  • 概述
    概述部分包含关于此下载所解决的问题的简短说明，以及此下载能够提供的好处。
  • 系统要求
    请始终检查系统要求部分，以确保您的计算机符合要求，并具有安装和运行此下载所需的空间。此部分可能还会列出必需或推荐的应用程序软件。
  • 说明
    请在单击下载按钮之前查看说明部分，以便您准备好执行任何必需的步骤。
  • 其他用户正在下载的内容
    列出已下载当前项目的用户所下载的其他项目。关于这些相关下载的信息会非常有帮助。
  • 其他信息
    提供与当前下载直接相关，或旨在帮助保持您的系统安全运行的更多信息，例如安全更新信息。
  • 相关资源
    这些链接指向与您已选择的下载相关的操作方法信息、新闻组、特殊产品和服务以及其他资源。

+如果不能完成下载，我应该怎么做？

  • 清除缓存
    如果您无法完成下载，则可能需要清除 Windows Internet Explorer 中的缓存，如以下步骤所述。请注意，此过程会略有不同，具体取决于您已安装的 Internet Explorer 版本。
    • 在 Internet Explorer 中的工具菜单上，单击Internet 选项，然后单击常规选项卡。
    • 在浏览历史记录部分，单击删除。
    • 在删除浏览的历史记录对话框的Internet 临时文件部分，单击删除文件，然后单击是（如果您要清空 Internet 临时文件的整个缓存）。
    • 在删除浏览的历史记录对话框的历史记录部分，单击删除历史记录。当系统询问您是否要删除已访问的网站历史记录时，请单击是。完成后，请单击关闭。
    • 在常规选项卡的浏览历史记录部分，单击设置。在设置对话框中，选择每次访问网页时。然后，将要用于Internet 临时文件文件夹的磁盘空间设置为大于所需下载的大小。接下来，单击查看文件按钮。
    • 在打开的Temporary Internet Files窗口中，打开编辑菜单并单击全选。按删除。当系统询问您是否要删除已选中的所有 Cookie 时，请单击是。单击确定按钮，直至返回常规选项卡，然后单击确定以关闭该对话框。
  • 禁用自动终止
    另一种导致不能完成下载的可能原因是：Internet 服务提供商 (ISP) 可能在 Internet 连接空闲一定分钟数之后自动终止连接。请与您的 ISP 联系以禁止在下载文件时自动终止连接。

+如何安装所下载的软件？

必须先安装您下载的任何软件，然后才能使用它。例如，如果您下载了一个安全更新但没有安装它，则该更新将不会为您的计算机提供任何保护。按照下面的步骤保存和安装下载：

  • 在下载详细信息页上，单击下载按钮。
  • 此时将打开一个对话框。单击保存按钮。
  • 系统将提示您在计算机上为下载选择一个位置。选择适当的位置，然后单击保存按钮。
  • 在完成文件下载后，您将看到第二个对话框。如果要立即安装文件，请单击打开按钮。这样将启动安装向导。
  • 如果不想立即安装文件，请单击关闭按钮。当您准备好安装文件时，双击文件即可启动安装向导。

注意：保存下载文件时，请确保记得文件的保存位置，以便在准备好安装时使用。建议在桌面上创建一个专门用于保存下载的特殊文件夹。按照下面的步骤创建这样一个下载文件夹：

  • 右键单击桌面上未使用的区域。
  • 指向新建，然后单击文件夹。
  • 为该文件夹指定名称，例如“My Downloads”。
